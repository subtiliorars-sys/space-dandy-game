## Smoke checklist (~3 min)

1. Open playtest hub → embedded game loads
2. Title → SPACE → patrol starts
3. Collect 3 orbs — combo text appears
4. Hit hazard — lives decrement, screen shake
5. Lose all lives — game over shows score + high score
6. Press C — CRT overlay toggles (HUD shows ON/OFF)
7. Resize browser — canvas scales (FIT)
