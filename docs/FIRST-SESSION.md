# First session — honesty brief

**Audience:** new players / playtesters  
**Purpose:** set expectations for the first Groove Patrol run

## What this game is

- Neon space arcade mini-game on Phaser 3 + TypeScript (groove orbs, hazards, combos).
- Fan homage with **original** characters/mechanics — **not** affiliated with official *Space Dandy* IP.

## What you will **not** see

- FOMO timers or streak-shame for leaving mid-run
- Pay-to-progress / loot-box framing
- Forced account for local `npm run dev`

## Suggested first minutes

1. `npm install` → `npm run dev`.
2. Collect a few groove orbs; try CRT toggle (C) once.
3. `npm run verify` before any PR.
4. Feature open: [PR #2](https://github.com/subtiliorars-sys/space-dandy-game/pull/2) (groove/mute + crossover fuel) — leave merge to Mark.
5. Do **not** itch-upload without Mark.
