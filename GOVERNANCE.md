# Governance — Space Dandy Game

- **No dark patterns:** no FOMO timers, no missable dailies, no loot boxes.
- **IP hygiene:** fan homage only — do not ship official Space Dandy logos, character art, or licensed audio without owner legal clearance.
- **Honest early access:** `playtest.html` and store copy must say what is and is not in the build.
- **Consent-based feedback:** playtest notes via GitHub Issues — no hidden telemetry.

Pricing / itch paid tier / Steam: owner approval before changes.
